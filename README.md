# Task Management

App de gerenciamento de tarefas com autenticação. Backend em Node.js + Express, frontend em React, banco SQLite.

## Stack

- **Backend:** Node.js, Express, TypeScript, better-sqlite3, JWT, bcryptjs
- **Frontend:** React, TypeScript, Vite, Axios

## Como rodar

```bash
# backend
cd backend
npm install
npm run dev

# frontend (em outro terminal)
cd frontend
npm install
npm run dev
```

Backend roda na porta 5000, frontend na 3000.

O banco SQLite é criado automaticamente ao iniciar o backend (não precisa configurar nada).
